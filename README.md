# redux-react-spboot

This is the sample TODO application with Redux and Spring Boot (Java).

###Usage

Install:
```
1. git clone https://github.com/onBass-naga/redux-react-spboot
2. cd redux-react-spboot/assets/scripts
3. npm install
4. cd ./../../
5. ./gradlew bootRun
6. Surf to localhost:8080
```
